Mis Conocimientos

Como programador, tengo la capacidad de crear sitios web funcionales y centrados en la experiencia del usuario. Estoy comprometido en seguir aprendiendo y ofrecer soluciones creativas y de calidad en el desarrollo web. Estos son los lenguajes que he aprendido.

HTML

HTML (HyperText Markup Language) es el lenguaje estándar para crear y estructurar contenido en la web. Es utilizado para definir la estructura y el significado semántico de una página web.

CSS

CSS (Cascading Style Sheets) es un lenguaje de estilo utilizado para describir la presentación y el diseño de un documento HTML. Con CSS, puedes controlar la apariencia visual de los elementos HTML, como colores, fuentes, márgenes, espaciado, diseño de página y efectos visuales.
